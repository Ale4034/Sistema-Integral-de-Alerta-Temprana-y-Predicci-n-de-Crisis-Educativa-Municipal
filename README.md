# Sistema Inteligente de Alerta Temprana para Crisis Educativa Municipal

Proyecto desarrollado para el concurso **Datos al Ecosistema 2026 – IA para Colombia**, organizado por el Ministerio TIC.

## Objetivo

Desarrollar un sistema basado en Inteligencia Artificial capaz de:

- Clasificar municipios según su nivel de riesgo educativo.
- Predecir indicadores de deserción y cobertura escolar.
- Generar alertas tempranas para apoyar la toma de decisiones.
- Visualizar los resultados mediante un dashboard interactivo.

## Tecnologías

- Python
- Pandas
- Scikit-Learn
- TensorFlow
- Streamlit
- Plotly
- Folium
- SHAP
